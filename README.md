## What This Project Does

An end-to-end pricing analytics system for a used car dealership — from raw transaction data to executive-ready insights. Combines SQL database design, advanced querying, Python analysis, and a machine learning price prediction model.

**Open `dashboard.html` in any browser to view the full interactive report.**

## SQL Highlights

**12 queries across 4 analytical themes:**

- **Revenue Intelligence** — Segment revenue by brand, model, condition with margin metrics
- **Forecasting** — 3-month rolling revenue average using `AVG() OVER (ROWS BETWEEN ...)`
- **Inventory Optimization** — Aging bucket analysis with profit-per-day-on-lot metric
- **Competitive Benchmarking** — Price variance vs. market average to surface revenue leakage
- **Performance Ranking** — Multi-metric salesperson scorecard using `RANK() OVER (...)`
- **Cohort Analysis** — ROI by vehicle age to guide acquisition strategy

---

## Price Prediction Model

Gradient Boosting Regressor trained on 520 transactions:
- **R² ≈ 0.94** — explains 94% of sale price variance
- **MAE < $800** — average prediction error under $800
- **Top features**: Acquisition cost → Vehicle age → Mileage → Make/Model
